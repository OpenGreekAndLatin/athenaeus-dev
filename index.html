Welcome to athenaeus-dev!

About

This repository hosts digital versions of public-domain editions of the works of Athenaeus of Naucratis, a Greek rhetorician and grammarian.

The XML files were produced by data entry company Digital Divide Data (DDD), who OCR'd, corrected and encoded each print volume in accordance with the latest EpiDoc standards. While adequate, more can and should be done to improve the XML (the -dev(elopment) suffix indicates the repository is open to improvement).
